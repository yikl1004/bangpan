주석사용

1. CSS 내 주석

■ CSS 페이지 기본주석

/****************************************/
/*	Name : PROJECT NAME                    : 프로젝트명(내용고정)
/*	PART : RESET STYLE                     : 해당메뉴 표시 (ex.메인의 경우 MAIN STYLE)
/*	Version : 1.0                          : 내용고정
/*	Author : 홍길동						 : 최초 작성자
/*	MODIFY : 김수정, 이수정, 박수정		     : CSS 파일 작업참여 구성원
/****************************************/

■ CSS 페이지내 수정/삭제시 주석

기존 소스 수정/삭제시 원본내용은 주석으로 백업처리

/* body {color:#fff; margin:0; padding:0;}
2015.01.01(수정/삭제일자) | 홍길동(수정/삭제자) | 디자인 변경으로 인한 기존 칼라값 변경(수정/삭제사유) */
body {color:#000; margin:0; padding:0;}

2. HTML 내 주석

■ 마크업을 위한 주석사용

<!-- 예제 안내 시작 -->
<div id="test">
	<div class="testDsc">예제안내</div><!--// div testDsc end -->
</div><!--// div test end -->
<!--// 예제 안내 끝 -->

■ 개발이나 퍼블리셔에게 내용 전달을 위한 주석사용

D : developer, P : publisher

<!-- D : 본 주석은 개발자를 위한 주석으로 개발 참고 후 삭제바랍니다.
		 개발에 전달해야되는 내용을 상세히 기재
         ex) 첫번째 li에 클래스 first 추가 -> <li class="first">
-->
<ul>
	<li class="first">첫번째 리스트</li>
    <li>두번째 리스트</li>
    <li>세번째 리스트</li>
</ul>

<!-- P : 본 주석은 퍼블리셔를 위한 주석으로 코딩작업 참고 후 삭제바랍니다.
		 퍼블리셔들에게 전달할 내용을 상세히 기재
         ex) 공통 샘플링 dl요소 확인 -> cm.common.css LINE.25 참고바람
-->
