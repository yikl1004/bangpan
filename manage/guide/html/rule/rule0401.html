■ form 요소를 포함하고 있는 경우 연관있는 입력문에 대해 그룹핑

  • 예시 1)
    입력폼 그룹핑 예제1 검색 
    <fieldset>
    <legend>입력폼 그룹핑 예제1</legend>
    <label for="exSel"></label>
    <select name="sample" id="exSel">
        <option value="">제목</option>
    </select>
    <a href="#" class="guideBtn">검색</a>
</fieldset>
  • 예시 2)
    입력폼 그룹핑 예제2
    입력폼 그룹핑 예제3 
    <fieldset class="guideEx">
    <legend>입력폼 그룹핑 예제2</legend>                            		
    <input type="radio" name="gender" id="exMan" checked><label for="exMan">남자</label>
    <input type="radio" name="gender" id="exWoman"><label for="exWoman">여자</label>
</fieldset>

<fieldset class="guideEx">
    <legend>입력폼 그룹핑 예제3</legend>                            		
    <input type="checkbox" name="age" id="age10" checked><label for="age10">10대</label>
    <input type="checkbox" name="age" id="age20"><label for="age20">20대</label>
    <input type="checkbox" name="age" id="age30"><label for="age30">30대</label>
    <input type="checkbox" name="age" id="age40"><label for="age40">40대</label>
</fieldset>

■ 입력폼 속성 정의 순서

  • 텍스트 박스 
    <input type="text" name="" value="" title="" class="" style="">
  • 이미지/버튼 
    <input type="image" name="" src="" alt="" class="">
  • radio/checkbox 
    <input type="radio" name="" id="" value="" class="" checked>

■

label은 명시적으로 사용.

<input type="checkbox" name="" id="ok" value="" checked=""> <label for="ok">동의</label>

■ overflow:scroll/overflow:auto 적용시 tabindex="0" 적용

  • 약관 또는 레이아웃 반영을 위해 엘리먼트에 overflow:scroll/overflow:auto 적용시 tabindex="0" 적용하도록 한다.
    • CSS 
      div.agreeBox {background:#f7f7f7; padding:10px; height:60px; border:1px #ddd solid; overflow-y:scroll;}
    • HTML 
      <div class="agreeBox" tabindex="0">
내용...
</div>
