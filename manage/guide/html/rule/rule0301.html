컨텐츠가이드

1. 페이지타이틀 작성순서

HTML문서에는 페이지 타이틀만 삽입. 메뉴뎁스는 개발모듈로 처리함.

■ 본문(일반) 페이지타이틀 작성

<title>페이지 타이틀 | 프로젝트</title>

■ 본문(글쓰기가 있는 페이지) 페이지타이틀 작성

<title>[신청/작성/글쓰기] 페이지 타이틀 | 프로젝트</title>

■ 팝업 페이지타이틀 작성

<title>[팝업] 페이지 타이틀 | 프로젝트</title>

2. 대체텍스트

  • <img>, <area>, <input type="image"> 요소는 반드시 alt="" 속성으로 대체 텍스트 제공 필수
  • 이미지+텍스트 링크 또는 비주얼 전경이미지와 같이 디자인 요소에 국한된 이미지의 경우 alt="" 빈값으로 처리 
    <img src="bg1.gif" alt="">
<a href="#" title="새창 띄움"><img src="logo.gif" alt=""> 개인정보보호(ePRIVACY)우수사이트</a>
  • 대체 텍스트가 길거나 대체 컨텐츠 마크업이 필요한 경우, class="hide" 혹은 :before(모바일 국한) 속성으로 처리. (style="display:none", "visibility:hidden" longdesc 속성 사용 불가)
  • 새창/팝업/설치파일/다운로드/동의 등 링크 부연설명 : title속성으로 대체텍스트 제공 
    <a href="#" title="새창 띄움">...<a>
<span>자동이체 약관동의</span> <label for="ok"><input type="checkbox" name="" id="ok" value="" checked="" title="자동이체 약관동의">동의</label>
  • <input type="text"> : <label> 미적용 시 title 속성 필수 
    <input type="text" name="" value="" size="" title="이름">
  • select 요소, textarea 요소 : title 속성 필수 
    <select title="생년월일 년도 선택">
<textarea cols="" rows="" title="내용">
  • 아이프레임 : title 속성 필수. 아이프레임의 용도를 설명하거나 개발구현을 위한 빈프레임일 경우에도 아래와 같이 title 속성을 제공해야 한다. 
    <iframe title="최근 공지사항 게시물">
<iframe title="Blank Frame / 빈프레임">

3. a 태그 사용시 주의사항

  • 새창 또는 사용자가 요구하지 않는 기능(레이어팝업, 인쇄옵션 등)을 포함하는 a 태그에는 반드시 target="_blank" 및 title="xxx 페이지 새창으로 열림" 표시 필수 
    <a href="pop.html" target="_blank" title="새창열림">...</a>
<a href="layer.html" title="레이어팝업열림">...</a>
<a href="print.html" target="_blank" title="인쇄옵션열림">...</a>
  • id 값 꼬는 javascript 값으로 이동할 경우 클릭시 다른곳으로 포커스가 이동하지 않도록 포커스 유지 처리가 되야 함 
    <a href="#tabMenu" onclick="return false;">...</a>
<a href="#" onclick="javascript:password('view', '86','qna');return false;">...</a>
