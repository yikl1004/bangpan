2. XHTML 문서 작성시 주의사항

  • DTD를 제외한 모든 엘리먼트와 애트리뷰트는 소문자로 작성한다.
    잘못된 사용 ex) 〈P Class=""〉〈/P〉, 〈a href="" onClick="" onMouseOver=""〉
  • 애트리뷰트값은 큰따옴표("")로 묶는다.
    잘못된 사용 ex) 〈img src=‘test.gif’ width=‘150’ height=’25’ alt=‘테스트’〉
  • entity 코드 사용 필수 http://www.w3schools.com/charsets/ref_html_entities_4.asp 참고
    잘못된 사용 ex) href=“board＆num=1” → board＆amp;num=1
  • html 페이지내에 자바스크립트 사용시 html 태그를 포함하고 있는 경우 //〈![CDATA[ , //]]〉 사용 필수.
    <script type="text/javascript">
	//');
	//]]>
</script>
  • DTD XHTML 1.0을 사용중이므로 반드시 종료태그를 선언한다.
    주의해야할 종료태그 앨리먼트 : area, br, col, frame, hr, img, input, link, meta, param
  • 의미있는 객체를 구분하기 위하여 코드 그룹간 1줄씩 빈 줄을 만드는 것은 허용한다. 단, 빈 줄의 간격은 1줄을 초과하지 않는다.
